# zakelijkinkomenberekenen.nl

Toetsinkomen calculators voor ondernemers (IB-ondernemer en DGA/BV) met NHG en reguliere varianten.

## Bestanden uploaden naar GitHub

1. Ga naar je repository op GitHub: `https://github.com/JOUW-NAAM/zakelijkinkomenberekenen`
2. Klik op **"uploading an existing file"** of **"Add file" > "Upload files"**
3. Sleep alle bestanden en mappen uit dit ZIP-bestand naar het uploadvenster
4. Klik op **"Commit changes"**

## Vercel koppelen

Zodra de bestanden op GitHub staan, gaat Vercel automatisch deployen.

## Domein koppelen

1. Ga in Vercel naar je project > **Settings** > **Domains**
2. Voeg `zakelijkinkomenberekenen.nl` toe
3. Vercel toont DNS-instellingen — voer deze in bij je domeinregistrar
